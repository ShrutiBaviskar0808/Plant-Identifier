# Montserrat Font Files Required

To complete the font setup, please download the following Montserrat font files from Google Fonts (https://fonts.google.com/specimen/Montserrat) and place them in this directory:

1. Montserrat-Regular.ttf
2. Montserrat-Medium.ttf  
3. Montserrat-SemiBold.ttf
4. Montserrat-Bold.ttf

These files are referenced in pubspec.yaml and the app theme.
